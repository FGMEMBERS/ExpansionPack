1. Installation¶

The FlightGear Expansion Pack can be installed by downloading a .zip file or by cloning the Git repository.

1.1. Dependencies¶

  • FlightGear 3.4

1.2. Installing from master.zip¶

To download the .zip file, download master.zip and then rename the folder fg-expansion-pack-master to ExpansionPack. Then copy the folder to FlightGear’s Aircraft folder.

1.3. Installing from the Git repository¶

Alternatively, you can clone the master branch of the Git repository. The repository can be found on Github at onox/fg-expansion-pack and can be cloned with:

$ git clone https://github.com/onox/fg-expansion-pack.git ExpansionPack
